# Методологическая база

## Основной материал текущей версии

Правила корпоративных методик и пороговые диапазоны перенесены из предоставленного пользователем материала:

- «Психологическое тестирование кандидатов», НОЦ «Бирюч».
- В таблице актуализаций документа указана дата 30.01.2025.
- Машиночитаемая версия правил находится в `app/methodology.py`.

Сам исходный PDF не включается в репозиторий. Это уменьшает риск нарушения прав на распространение и случайной публикации служебных материалов.

## Правила, имеющие приоритет в проекте

- Интерпретация строится по целостному профилю и сочетаниям показателей.
- При L/F/K выше 80 профиль помечается как ненадежный; диапазон 70-80 требует осторожности.
- По явному требованию владельца проекта пятая шкала интерпретируется в одном направлении для мужчин и женщин: высокое значение в сторону феминизированности, низкое в сторону мускулинности.
- Высокое значение социальной интроверсии трактуется как интровертированность, низкое как экстравертированность.
- Корпоративные проценты классифицируются только по диапазонам методики. Они не называются популяционными перцентилями.
- Итоговый текст не содержит клинических терминов и технических названий шкал.

## Контроль источников

Добавлять внешний источник следует только после проверки версии, применимости к конкретной адаптации теста и лицензионных условий. Ссылки без полного текста, недоступные материалы и обобщения из поисковой выдачи не должны становиться правилами программы.

Любое изменение порогов требует:

1. указать документ, редакцию и страницу;
2. добавить тест на граничные значения;
3. обновить `METHODOLOGY_VERSION`;
4. проверить изменение на обезличенных контрольных профилях;
5. провести содержательную проверку квалифицированным психологом.
