# 📚 Источники и методологическая база Psycho Portrait

## Основная литература

### СМИЛ (адаптация MMPI)
1. **Собчик Л.Н.** — «СМИЛ (MMPI). Стандартизированный метод исследования личности» (изд. 3-е, 2020).
   - Ввела модифицированные (поведенческие) названия шкал: 1 (Hs) — невротический сверхконтроль,
     2 (D) — пессимистичность, 3 (Hy) — эмоциональная лабильность, 4 (Pd) — импульсивность,
     6 (Pa) — ригидность, 7 (Pt) — тревожность, 8 (Sc) — индивидуалистичность,
     9 (Ma) — оптимизм и активность, 0 (Si) — социальная интроверсия.
   - Используется в нашем промпте.

2. **Березин Ф.Б., Мирошников М.П., Соколова Е.Д.** — «Методика многостороннего исследования
   личности. Структура, основы интерпретации, некоторые области применения» (3-е изд.).
   - Базовая методика ММИЛ (377 вопросов) — адаптация MMPI для русскоязычной популяции.
   - 3-е издание дополнено главой по интерпретации 16PF Кеттелла + совместная интерпретация ММИЛ+16PF.

3. **Демкин А.Д.** — статья «Расшифровка и интерпретация результатов теста MMPI (СМИЛ)»,
   https://onkto.ru/blog/psychometry/rasshifrovka-i-interpretatsiya-rezultatov-testa-mmpi-smil
   - 8-шаговый алгоритм интерпретации.
   - Правила кодирования профиля (T > 69 для СМИЛ/ММИЛ, T > 64 для MMPI-2).
   - Предупреждение о стигматизации.

4. **Psylab.info** — «Стандартизированный многофакторный метод исследования личности»,
   https://psylab.info/Стандартизированный_многофакторный_метод_исследования_личности
   - Тексты опросников (мужской, женский, подростковый), профильные листы, ключи.
   - Описание интерпретации шкал.

### 16PF Кеттелла
5. **Psyfactor.org** — «Тест 16PF Кеттелла. Психологическая интерпретация факторов»,
   https://psyfactor.org/lib/cattell-16pf.htm
   - Подробное описание всех 16 факторов, шкала стэнов 1-10.

6. **HRDirector** — «Тест Кеттелла 16PF: многофакторное исследование личности»,
   https://www.hr-director.ru/article/66940-oprosnik-kettella-metodika-issledovaniya-17-m11
   - Применение в HR-практике.

### DISC
7. **Stratoplan** — «Пройти тест DISC онлайн бесплатно»,
   https://stratoplan-school.com/disc
   - Расшифровка 4 осей (D, I, S, C).

8. **Sekretariat.ru** — «Зачем нужен тест DISC и как интерпретировать результаты»,
   https://www.sekretariat.ru/article/211617-test-disc-na-tipy-lichnosti

### HOLLAND
9. **Wikipedia / Holland Codes (RIASEC)** — базовое описание 6 типов.

### Big Five
10. **NCBI/PMC** — «Big Five personality traits in the workplace» (2023),
    https://pmc.ncbi.nlm.nih.gov/articles/PMC10089283

## Этические ограничения

- Программно-сгенерированный текст — **черновик**, а не окончательное заключение.
- Финальное слово — за психологом с профильным образованием.
- Не использовать для кадровых решений без верификации специалистом.
- Не передавать третьим лицам без согласия обследуемого (ФЗ-152 о персональных данных).
- Методики измеряют предпочтения и актуальные состояния, а не «истинную» личность.
