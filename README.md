## ✨ Что это такое

**Psycho Portrait** — инструмент для психолога-практика, который превращает сухие баллы по тестам в профессиональную психологическую характеристику сотрудника.

```
PPTX с результатами тестов
         ↓
   парсер (python-pptx + regex)
         ↓
  структурированный профиль
         ↓
   LLM (DeepSeek / GLM / OpenAI)
         ↓
  готовая характеристика в Markdown
```

> ⚠️ **Это черновик-ассистент**, не замена психологу. Финальное заключение — за специалистом.

## 🧪 Методики

| Методика | Что считывается | Кол-во шкал |
|----------|----------------|-------------|
| **16PF Кеттелла** | 16 первичных факторов (стэны 1-10) | 16 |
| **Big Five (BFI / NEO-PI-R)** | O, C, E, A, N (баллы 0-100) | 5 |
| **СМИЛ / ММИЛ / MMPI** | Валидность L/F/K + 10 базисных (T-баллы) + код профиля + тип | 13 + служебные |
| **Корпоративные шкалы** | «Джентльмен», «Здравомыслие» (ЭФКО) | 2 |
| **DISC** | D, I, S, C (баллы 0-100) | 4 |
| **HOLLAND** | RIASEC + 3-буквенный код проф. типа | 6+1 |
| **MBTI** | 4 дихотомии → 16 типов | 4 |
| **Амтхауэр** | IQ + субтесты | 1+9 |

## 📚 Методологическая база

Интерпретация построена на работах:

- **Собчик Л.Н.** — «СМИЛ (MMPI). Стандартизированный метод исследования личности» (3-е изд., 2020).
  Ввела модифицированные поведенческие названия шкал вместо клинических (не «шизофрения», а «индивидуалистичность»; не «паранойя», а «ригидность»).
- **Березин Ф.Б., Мирошников М.П., Соколова Е.Д.** — «Методика многостороннего исследования личности» (3-е изд., 377 вопросов, глава по совместной интерпретации ММИЛ+16PF).

### Правила интерпретации СМИЛ (по Собчик + Демкину)
- По **целостному профилю и коду**, а не по отдельным шкалам
- Код надёжен при **T > 69** (СМИЛ/ММИЛ) или **T > 64** (MMPI-2)
- Различие между пиками ≥ **5 T-баллов**
- **5 сфер анализа:** личностные черты → проблемные зоны → ведущие потребности → когнитивные искажения → профпригодность

Полный список литературы и GitHub-проектов — в [docs/sources.md](docs/sources.md).

## 🚀 Быстрый старт

```bash
# 1. Клонируй
git clone https://github.com/Saborrr/psycho-portrait.git
cd psycho-portrait

# 2. Виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. Переменные окружения
cp .env.example .env
# Открой .env и вставь свой LLM_API_KEY

# 4. Старт
uvicorn app.main:app --reload --port 8000
```

Открой **http://localhost:8000** → drag&drop PPTX → на выходе характеристика.

### Или через Docker

```bash
docker build -t psycho-portrait .
docker run -p 8000:8000 --env-file .env psycho-portrait
```

## 📡 API

| Метод | Путь | Что делает |
|-------|------|-----------|
| `GET` | `/` | Web UI (drag&drop) |
| `GET` | `/health` | Health check |
| `POST` | `/api/parse` | Только парсинг PPTX → JSON (для отладки шаблона) |
| `POST` | `/api/generate` | Полный цикл: парсинг + LLM → характеристика |
| `POST` | `/api/raw-text` | Только текст из PPTX (для отладки парсера) |

### Пример: `/api/generate` через curl

```bash
curl -X POST -F "file=@samples/sample_employee.pptx" \
     -F "style=default" \
     http://localhost:8000/api/generate
```

Ответ: JSON с `profile` (извлечённые данные) + `characteristics_markdown` (готовая характеристика).
